# v0.1.6 Scoped Serve Ports Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [x]`) syntax for tracking.

**Goal:** Make `ohbaby serve` safe for the v0.1.6 single-project web UI by scoping server state to the current project root, reusing same-scope servers, auto-assigning a free port for different scopes, explaining explicit port conflicts, and opening the browser to the final URL.

**Implementation Status:** Completed in this branch. The original task checklist is retained as the implementation record and marked complete.

**Architecture:** Add a small scoped serve layer inside `ohbaby-server/runtime/daemon`: resolve a git-root-aware scope, derive per-scope state paths, probe existing state for reuse, and start a foreground server with default-port fallback only when `--port` was omitted. Keep browser opening in `ohbaby-cli` as an injectable runtime dependency.

**Tech Stack:** TypeScript, Node.js, Hono node server, Vitest, existing `ohbaby-agent` project resolver and SQLite-backed persistent backend.

- [x] **Step 3: Implement scope resolver**

Create `scope.ts`:

```ts
import { realpath } from "node:fs/promises";
import { join, resolve } from "node:path";
import { Project } from "ohbaby-agent";

export interface DaemonScope {
  readonly scopeRoot: string;
  readonly pidFilePath: string;
  readonly stateFilePath: string;
}

export interface ResolveDaemonScopeOptions {
  readonly workdir?: string;
}

async function canonicalDirectory(path: string): Promise<string> {
  try {
    return await realpath(path);
  } catch {
    return resolve(path);
  }
}

export async function resolveDaemonScope(
  options: ResolveDaemonScopeOptions = {},
): Promise<DaemonScope> {
  const workdir = await canonicalDirectory(options.workdir ?? process.cwd());
  const projectRoot = await Project.getProjectRoot(workdir);
  const scopeRoot = projectRoot ? await canonicalDirectory(projectRoot) : workdir;
  const serverDir = join(scopeRoot, ".ohbaby", "server");
  return {
    pidFilePath: join(serverDir, "daemon.pid"),
    scopeRoot,
    stateFilePath: join(serverDir, "daemon-state.json"),
  };
}
```

```ts
import { daemonAuthHeader } from "../../auth/token.js";
import type { DaemonState } from "./types.js";

export type DaemonHealthCheck = (state: DaemonState) => Promise<boolean>;

export const fetchDaemonHealth: DaemonHealthCheck = async (
  state,
): Promise<boolean> => {
  if (!state.host || !state.port || !state.authToken) {
    return false;
  }
  try {
    const response = await fetch(`http://${state.host}:${String(state.port)}/api/health`, {
      headers: { authorization: daemonAuthHeader(state.authToken) },
    });
    return response.ok;
  } catch {
    return false;
  }
};
```

```ts
async function openUrl(url: string): Promise<void> {
  const { spawn } = await import("node:child_process");
  const command =
    process.platform === "darwin"
      ? "open"
      : process.platform === "win32"
        ? "cmd"
        : "xdg-open";
  const args =
    process.platform === "win32" ? ["/c", "start", "", url] : [url];
  await new Promise<void>((resolve, reject) => {
    const child = spawn(command, args, {
      detached: true,
      stdio: "ignore",
    });
    child.once("error", reject);
    child.once("spawn", () => {
      child.unref();
      resolve();
    });
  });
}
```
