# Colors 颜色定义

> 待撰写：颜色定义详细设计

## 颜色分类

### 文本颜色

- `text.primary`: 主要文本 (#FFF)
- `text.secondary`: 次要文本 (#A0A0A0)
- `text.accent`: 强调文本 (#00BFFF)

### 消息颜色

- `message.user`: 用户消息 (#00FF00)
- `message.assistant`: AI 响应 (#00BFFF)
- `message.system`: 系统消息 (#FFFF00)
- `message.error`: 错误消息 (#FF0000)

### 工具状态颜色

- `tool.pending`: 等待中 (黄色)
- `tool.running`: 执行中 (蓝色)
- `tool.completed`: 已完成 (绿色)
- `tool.error`: 错误 (红色)
- `tool.aborted`: 已中止 (灰色)

### UI 颜色

- `ui.border`: 边框 (#444)
- `ui.background`: 背景 (#1A1A1A)
- `ui.highlight`: 高亮 (#333)
